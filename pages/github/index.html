Instalação do Git e uso com GitHub

Instalação do Git

git --version
ℹ️ Você deve ver a versão do Git instalada, como "git version 2.40.0".

Configuração inicial do Git

git config --global user.name "Seu Nome"
git config --global user.email "seuemail@exemplo.com"
git config --list

Crie um repositório no Github

ℹ️ O colaborador receberá um email com convite para participar do seu repositório e deve aceitar o convite para acessá-lo.

Clonando o repositório para sua máquina

git clone https://github.com/seu-usuario/meu-primeiro-projeto.git
cd meu-primeiro-projeto

Primeiro commit

git add .
git commit -m "Initial commit"
git push origin main
ℹ️ Agora você pode acessar o seu repositório no GitHub e verificar se os arquivos foram enviados com sucesso.

Atualizando seu repositório local

git pull origin main
⚠️ Lembre-se de sempre utilizar esse comando antes de commitar uma alteração.

Conclusão

Parabéns! Você aprendeu os comandos básicos do Git e como usar o GitHub para controle de versão dos seus projetos. Continue praticando esses comandos para se familiarizar com o fluxo de trabalho do Git.
